اپلیکیشن مدرن برای خرید گل که با استفاده از **Flutter** طراحی شده است.

> **وضعیت پروژه:** این پروژه در حال حاضر یک نسخه نمایشی (UI/UX Demo) است و تمرکز اصلی آن بر طراحی رابط کاربری و پیاده‌سازی انیمیشن‌های نرم بوده است.

---

##  ویژگی‌های پروژه (Features)
* **طراحی واکنش‌گرا:** رابط کاربری منعطف برای سایزهای مختلف موبایل.
* **انیمیشن‌های Hero:** جابجایی جذاب بین لیست گل‌ها و صفحه جزئیات.
* **مدیریت وضعیت (State Management):** استفاده از **Provider** برای منطق سبد خرید (در حافظه موقت).
* **ساختار بهینه:** مخزن گیت‌هاب تمیز و ۱۰۰٪ Dart.

---

## ابزارهای استفاده شده (Technologies)
* **Flutter & Dart** (Core)
* **Provider** (State Management)
* **Material Design 3** (UI System)

## نکاتی درباره این نسخه (Current Status)
* این نسخه فاقد دیتابیس (Local Database) است، بنابراین با خروج از برنامه، اطلاعات سبد خرید ذخیره نمی‌شود.
* برخی دکمه‌ها در صفحه Home صرفاً جنبه نمایشی دارند و هنوز به صفحات دیگر لینک نشده‌اند.
* **نسخه APK:** این پروژه شامل تنظیمات بیلد اندروید (Gradle/Kotlin) برای دریافت خروجی نهایی است.

---

##  English Summary
This is a sleek Flower Shop UI kit built with Flutter. It focuses on clean UI, smooth Hero animations, and state management using Provider.
* **Note:** This is a UI demo; persistent storage and some internal links are not yet implemented.
